Other Development API Resources

World Bank API - Access World Bank indicator, projects and financial data via their API

AidInfo - Explore the International Aid Transparency Initiative (IATI) Registry of aid-spending data by using their API, their IATI Explorer XML Toolkit or their IATI Codelist API.

Guardian Open Platform - Search the large global development section of this UK news source for the latest news and commentary from around the world.
